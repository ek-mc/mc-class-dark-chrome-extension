# Privacy Policy

MC-Class Dark Theme does not collect, store, process, or transmit personal data.

## What the extension does
- Applies local CSS/JS styling on pages under `mc-class.gr` to provide a dark theme.

## What the extension does NOT do
- No analytics or telemetry
- No tracking cookies
- No external API calls for user data
- No sale or sharing of user data
- No remote code execution

## Permissions rationale
The extension requests host access for `mc-class.gr` only, solely to read/modify page content needed to apply the dark theme.
