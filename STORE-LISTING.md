# Chrome Web Store Listing Copy

## Name
MC-Class Dark Theme

## Short description
Dark theme for mc-class.gr with improved readability.

## Detailed description
MC-Class Dark Theme applies a clean, high-contrast dark interface to mc-class.gr.

Features:
- Dark palette with better text contrast
- Component-specific styling (LessonBox, AccordionCard, notifications, absences, menus)
- Select2 and pagination readability improvements
- Navbar toggler visibility fix
- Replaces default NoPhoto placeholders with an inline SVG dark placeholder

Permissions:
- Access limited to mc-class.gr domain only
- Required only to apply visual styling on those pages

Privacy:
- No analytics
- No tracking
- No personal data collection or transmission
- No remote code execution

Support:
https://github.com/ek-mc/mc-class-dark-chrome-extension
